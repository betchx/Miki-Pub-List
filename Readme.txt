東工大三木教授の論文リスト プロジェクト

JSCEの論文公開ページ，JSTAGEからダウンロードしたpdfにリンク付けされた
一覧を作成するというプロジェクト．

三木教授退職記念事業の一環として立ち上げたもの．


///////////////////
詳細等メモ
///////////////////

・作成方法
JabRefで文献を管理し，HTMLにエクスポートする形で進行．

・特殊処理：古い土論ではNo.をVol.として取り扱う
 かつての土論はVolumeがなく，No.のみというイレギュラーな形であるため，文献管理上問題となっていた．
 すなわち，通常articleではVolumeが必須フィールドであるため，ここがブランクだとエラーが発生してしまう．そのためNo.のみでは管理が難しい.
 しかし，よく見てみると，各号でページ番号が独立しており，実質上土論のNo.はVol.と等価である．したがって，No.をVolumeとして扱うことで，前述の問題を回避することとした．

・PDFのファイル名の命名規則
　発行年，巻，最初のページ，（和文）タイトルをアンダーバーで連結して拡張子をつけたもの．
　このうち，発行年・巻・ページは必須だが，タイトルはオプションである．

・PDFのリネーム
　鈴木助教がDLしたファイルからのリネームスクリプト(rename_all.sh)を作成済み．
　ただし，不完全でfix_name.shでの再変換が必要．
　将来的には公開サイトからDLしたファイルをリネームできる様にする予定
